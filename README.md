# website-security
Wordpress Website Security Points

Here's a polished and grammatically correct version:

1. **Create the MU Plugins Folder:** If it does not already exist, create a folder named **`mu-plugins`** inside the **`/wp-content/`** directory.
2. **Upload the Plugin File:** Upload the **`block-bad-http-hosts.php`** file to the **`/wp-content/mu-plugins/`** folder.
3. **Update the Root `.htaccess` File:** Add or update the root **`.htaccess`** file using the contents of the **`root-htaccess`** code file.
4. **Update the Plugins `.htaccess` File:** Add or update the **`.htaccess`** file inside the **`/wp-content/plugins/`** directory using the contents of the **`plugins-htaccess`** code file.
5. **Update the Uploads `.htaccess` File:** Add or update the **`.htaccess`** file inside the **`/wp-content/uploads/`** directory using the contents of the **`uploads-htaccess`** code file.
